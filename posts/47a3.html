前言

记载并引荐一些自己开发学习进程中常用的一些插件快捷键协助开发

关于vscode

插件

Markdown Preview Enhanced ： 本地Markdown文档预览插件。

react开发

Reactjs code snippets

react-redux-es6-snippets

vue开发

Vue语法高亮

快捷键

打开命令面板 cmd + shift + p / F1
打开设置页面 cmd + ,
删除一行 cmd + E(自定义)
添加书签 cmd + option + k
代码格式化 control + opiton + B
打开一个新窗口： Ctrl+Shift+N
关闭窗口： Ctrl+Shift+W
Ctrl+C 、 Ctrl+V 复制或剪切当前行/当前选中内容
整个文件夹中查找 Ctrl+Shift+F
全屏：F11

修改默认快捷键

File -> Preferences -> Keyboard Shortcuts，或者：Alt+F -> p -> k

修改 keybindings.json：

// Place your key bindings in this file to overwrite the defaults
[
    // ctrl+space 被切换输入法快捷键占用
    {
        "key": "ctrl+alt+space",
        "command": "editor.action.triggerSuggest",
        "when": "editorTextFocus"
    },
    // ctrl+d 删除一行
    {
        "key": "ctrl+d",
        "command": "editor.action.deleteLines",
        "when": "editorTextFocus"
    },
    // 与删除一行的快捷键互换
    {
        "key": "ctrl+shift+k",
        "command": "editor.action.addSelectionToNextFindMatch",
        "when": "editorFocus"
    },
    // ctrl+shift+/多行注释
    {
        "key":"ctrl+shift+/",
        "command": "editor.action.blockComment",
        "when": "editorTextFocus"
    },
    // 定制与 sublime 相同的大小写转换快捷键，需安装 TextTransform 插件
    {
        "key": "ctrl+k ctrl+u",
        "command": "uppercase",
        "when": "editorTextFocus"
    },
    {
        "key": "ctrl+k ctrl+l",
        "command": "lowercase",
        "when": "editorTextFocus"
    }
]

关于Chrome

插件

Proxy SwitchyOmega
轻松管理和切换多个代理设置

Hack bar
HackBar 工具包

shodan
Shodan插件网站的托管位置（国家/地区，城市），谁拥有IP和哪些其他服务/端口处于打开状态。

Wappalyzer
在信息搜集进程中，Wappalyzer可以剖析目的网站所采用的平台构架、网站环境、服务器配置环境、JavaScript框架、编程语言等参数。

ScanAnnotation
通过注释发现一些利用的内容

谷歌访问助手
这是一款免费FQ插件，轻松访问谷歌搜索，谷歌应用商店。一些不想花钱的，可以用一下

The Great Suspender
可以自动休眠暂时不用的选项卡以节省内存的开支
Adblock
屏蔽了几乎所有的烦人的弹出广告、Banner广告、图片广告等。直接带来的结果是：页面干净，加载速度变快、电量节省等。
JSONView
自动帮你排版json数据
tampermonkey
Tampermonkey「油猴」是一款免费的浏览器扩展和最为流行的用户脚本管理器。

关于火狐

插件

FoxyProxy Standard
FoxyProxy是一个初级的代理管理工具，它完全替代了Firefox无限的代理功用。可自定义配置代理并疾速完成代理切换。

Wappalyzer
在信息搜集进程中，Wappalyzer可以剖析目的网站所采用的平台构架、网站环境、服务器配置环境、JavaScript框架、编程语言等参数。

HTTP Header Live
此插件可捕捉网页中加载的全部网页流量数据包，点击任一数据包可对恳求包停止编辑与重放。
